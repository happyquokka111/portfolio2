Accessibility

I agree that many of the problems WAVE found makes it difficult for users to use the webpage. Firstly, there are several points of low contrast, so a user who has a difficult time seeing could not see certain elements of the page. WAVE also found multiple points of the redundant title text, which clutters the page and makes it difficult to navigate. Lastly, there are some headers and missing form labels, which can make understanding parts of the page confusing.
